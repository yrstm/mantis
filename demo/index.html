Mantis → agent Markdown

Converts readable web content to clean Markdown: articles, docs, newsletters, and reference pages.

Mantis captures the rendered DOM you can see and turns the main content into structured, token-efficient Markdown for an agent.

Use the extension for live pages: click the Mantis toolbar action, or select text first and click it to convert the selected DOM range.

Nothing leaves the page.

What changes for an agent

Browser copy-paste brings navigation, sidebars, and footer text. Mantis keeps the content and adds the metadata an agent can use to cite, dedupe, and judge capture quality.
