Mantis → Markdown

Converts the readable content of any web page to clean Markdown: articles, docs, newsletters, and reference pages.

Drag this to your bookmarks bar, then click it on any page you are reading:

Markdown this page

Mantis runs inside the page (the DOM you see, not the HTML the server sent) and shows the Markdown in an overlay with a copy button. Select text first to convert just the selection. Keep npm run demo running; it serves your working copy of mantis.js, uncached. Nothing leaves the page.

If clicking does nothing and no alert appears: Safari blocks https pages from loading localhost scripts. Chrome and Firefox allow 127.0.0.1. Use the paste fallback below.

No bookmarklet? Paste the page

Some sites' Content-Security-Policy blocks injected scripts. Follow the steps below to extract via paste instead.

1. Copy the page HTML

Run this in the devtools console on the page you want to convert:

copy(JSON.stringify({url: location.href, html: document.documentElement.outerHTML}))
