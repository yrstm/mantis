Drag this to your bookmarks bar, then click it on any page you are reading:

Markdown this page

Mantis runs inside the page — the DOM you see, not the HTML the server sent — and an overlay shows the Markdown with a copy button. Keep npm run demo running; it only serves your working copy of mantis.js. Nothing leaves the page.

No bookmarklet? Paste HTML

Some sites' Content-Security-Policy blocks injected scripts. Copy the page's HTML instead (devtools → copy outerHTML) and convert it here — same engine, via Mantis.fromHTML, entirely in this tab.
